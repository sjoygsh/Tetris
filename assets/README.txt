TETRIS GAME ASSETS
==================

This folder should contain the following files:

REQUIRED:
  font.ttf     — Any TrueType font file for text rendering
                  You can download a free one (see instructions below)

OPTIONAL (sound effects):
  move.wav     — Short click/tap sound for piece movement
  rotate.wav   — Rotation sound effect
  drop.wav     — Hard drop sound effect
  clear.wav    — Line clear sound effect
  gameover.wav — Game over sound effect
  music.ogg    — Background music (OGG format)

HOW TO GET A FONT:
  1. Download any .ttf font from https://fonts.google.com
     (e.g., "Roboto Mono", "Press Start 2P", "VT323")
  2. Rename the file to "font.ttf"
  3. Place it in this folder

The game will run without sound files but REQUIRES font.ttf to display text.
